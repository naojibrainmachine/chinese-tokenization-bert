# chinese-tokenization-bert
This was a Chinese tokenization which consist of bert.

直接运行train_bert_tokenize.py就能进行训练。  
运行test_bert_tokenize.py就能进行测试。 

测试准确率：74%  

关键库：
tensorflow 2.4
